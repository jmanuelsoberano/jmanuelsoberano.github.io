# Bienvenido

Este es el primer artículo del blog después de la migración a Astro.

## ¿Por qué Astro?

Astro es un framework moderno para construir sitios web rápidos, enfocados en contenido.

### Ventajas principales

- **Performance**: Output estático por defecto
- **Markdown/MDX**: Contenido como código
- **Islands Architecture**: JavaScript solo donde se necesita
- **SEO**: Meta tags y sitemap automáticos
